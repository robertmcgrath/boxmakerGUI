About BoxmakerGUI from the Champaign Urbana Community Fab Lab

 

The BoxmakerGUI program is a wrapper that invokes a modified version of ‘boxmaker’ from rahulbot.com.

 

The modified version of boxmaker includes several changes to make the software work well for the CUCFL.  The changes include four new parameters to the command line, with associated code.  These are:

 

·      “hide text” – suppress the descriptive text printed by boxmaker.  This text sometimes gets in the way of the outline of the box.

·      “epilog line width” – sets the lines to cut to .001in., as needed by the CUCFL Epilog laser.

·      “compact” – attempts to lay out the box in a compact area on the page, to reduce wasted material

·      “portrait/landscape” – hint to the compact layout to favor ‘tall’ or ‘wide’ layout.

 

The modified version of boxmaker is available free under a Creative Commons license.

 

The original boxmaker software is available from:  https://github.com/rahulbot/boxmaker.  The boxmaker code is deployed for public use at: http://boxmaker.rahulbotics.com/

 

boxmakerGUI is Java application which calls the modified boxmaker.  BoxmakerGUI lets the user enter the parameters of the box, and inspect the result in a preview.
