Boxmaker Parameters

 

Height, Width, and Depth define the size of the box.  These values must be given in millimeters (mm).

 

 Slot Width and Slot Depth define the tabs along the edges (in millimeters).

 

For a smooth fit, Slot Depth should be the same as the thickness of the material. 

 

Slot Width is the width of the tabs (in millimeters). 

 

 

The Cut Width (mm) is an adjustment to allow for the material removed by the laser.

 

This setting depends on the specific material used.  Also, it may be adjusted to

control the tightness of the press fit.

 

The drop down list has several materials for which there are suggested values.

By default, the program lays out the pieces “landscape” mode, wider than tall.

Portrait will layout more vertically.

 

Saving and Loading Parameters

 

The Save/Load feature allows you to save the settings for a projects and reload them later, or on another machine.  The output file is portable, and can be used on any machine, and can be emailed or shared via a URL.

 

The Save button writes out the current settings into an XML file.  The user is asked for the name of the file to save to.

 

The Load button reads an XML file selected by the user, and sets all the settings as they were saved.
